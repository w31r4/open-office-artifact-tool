# PDF

PDF is the file-type wrapper plugin for PDF artifact workflows.

This plugin is bundled into the primary runtime.

## Included Skills

- `PDF`: read, create, render, verify, and extract content from PDF files.

## Discoverability

Use this plugin for PDF-oriented terms from the file-type naming model: PDF, PDFs, render PDF, review PDF, extract PDF, pdfplumber, pypdf, reportlab, and `.pdf`.

## Source And Copy Flow

This plugin package is generated from `lib/artifacts/skills` and copied into `plugins/pdf` through `lib/artifacts/skills/mapping.json`.
